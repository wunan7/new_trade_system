## 2.5 核心发现

### V2 → V3 的改善来源

1. **事件驱动策略的爆发式增长**：信号从 8 次增至 574 次。龙虎榜事件驱动策略捕获的是机构短期集中买入的个股，这类股票在买入后短期内（5-10天）继续上涨的概率较高，显著提升了平均单笔盈利（+11.98% → +14.15%）。

2. **动量策略占比进一步下降**：从 59.3% 降至 38.4%。在震荡/熊市中，更少的动量交易意味着更少的"追涨杀跌"亏损，这是最大回撤进一步收窄（13.34% → 12.08%）的主要原因。

3. **多策略协同信号增加**：multi 信号从 47 次增至 65 次（+38%），说明事件驱动策略的加入增加了策略间的交叉验证，当多个策略同时看好一只股票时，该信号的置信度和后续表现通常更好。

4. **波动率基本不变**：年化波动率从 10.74% 微降至 10.67%，说明事件驱动策略的加入没有引入额外风险，纯粹是收益端的改善。
