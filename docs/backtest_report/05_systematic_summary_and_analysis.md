# 系统总结：MVP回测排雷与数据补全纪实 (2026-04-05)

## 1. 风控死锁与 0 交易之谜
在执行第一版 3 年全量回测时，我们遭遇了系统只在 2023 年跑了 10 个月就彻底停止交易的诡异现象（全年收益 +0.00%）。

**排雷结果**：
这源于 `DrawdownMonitor` 中一个极端严厉的“熔断”设计。原代码规定：一旦组合历史最大回撤跌破 **8% (YELLOW级别)**，就禁止开启任何新仓位。在真实长周期熊市中，稍微随大盘下潜就会触及 8%，一旦禁止买入，全转现金，组合净值就被永久锚定在这个水位，回撤永远保持 8% 以上，导致“死锁”。
**修正方案**：
放宽了各级风控阈值，保留了“限制仓位上限”的功能，而非“直接禁止买入”，从而让策略有机会在低位重启自我修复。
- YELLOW: 12% (限仓80%)
- ORANGE: 15% (限仓50%)
- RED: 20% (限仓20%)

## 2. 价值策略的“消失”与数据缺失链
当系统死锁被解除后，三年的总交易次数恢复至 6930 次，但我们发现了第二个致命问题：
主打低估值、高股息的防守型“价值策略”在 3 年内**仅仅触发了 13 次买入**！整个熊市中，系统 84% 的仓位竟然是靠量价追涨的“动量策略”在支撑。

**排雷结果**：
回溯到底层的 `finance` 数据库发现，`stock_valuation`（包含 PE/PB/股息等关键估值指标的表）在 **2023-01 至 2025-06** 期间完全没有被采集。没有 PE 和 PB 数据，因子引擎自动过滤掉了全市场的所有股票，导致“价值策略”在整整两年的熊市期间被强制“静音”。

**修正方案**：
将缺失字段清单交由数据采集组处理。数据采集组已利用接口补齐了 2023 至今每日 5000+ 股票的滚动市盈率（pe_ttm）、市净率（pb）和流通市值（circulating_market_cap）。

## 3. 资金面因子的“曲线救国”
由于上游 `finance_data` 采集脚本无法获取到 `north_net_buy`（北向资金绝对净买入额），导致这三年来该字段为全 NULL，相关的北向因子彻底失效。

**修正方案**：
利用已有的 `north_hold_pct`（北向持仓比例）字段进行近似替代。
通过重构 `calc_north_flow_chg` 因子函数：
$$ \text{北向买入强度} = \frac{ \text{今日持仓比例} - \text{N日前持仓比例} }{100} $$
这从数学本质上等同于“买入净额占总流通股本的比例”，无需依赖绝对金额即可精准还原资金流入的动能。

## 4. 下一步：见证完全体的威力
目前，利用最新补全的历史估值数据，后台正在执行全量因子重算（`recompute_factors.py`）。

届时，**动量、成长、价值、事件** 四大策略将在完整数据的驱动下，根据市场状态（BULL/BEAR/NEUTRAL）实现均衡轮动。系统有望大幅收窄前期因单靠“动量”硬抗熊市所造成的 -19.86% 亏损，跑出平滑且战胜基准的净值曲线。
