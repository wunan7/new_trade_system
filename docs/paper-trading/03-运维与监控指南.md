# 运维与监控指南

## 一、定时任务

系统配置了两个 Windows 定时任务，按顺序执行：

| 任务名 | 时间 | 脚本 | 作用 |
|--------|------|------|------|
| `FinanceData_DailyUpdate` | 每天 19:00 | `finance_data/scripts/run_daily_update.bat` | 更新行情、估值、资金流向等原始数据 |
| `TradingSystem_DailyPipeline` | 每天 19:30 | `new_solution/scripts/run_trading_daily.bat` | 因子计算→信号→风控→模拟成交 |

非交易日（周末、节假日）两个任务都会自动跳过。

## 三、查看持仓与绩效

### 当前持仓
```sql
SELECT code, open_date, open_price, shares, strategy_source,
       stop_loss_price, take_profit_price
FROM portfolio_positions
WHERE status = 'open'
ORDER BY open_date DESC;
```

### 已平仓记录（含盈亏）
```sql
SELECT code, open_date, close_date, open_price, close_price,
       close_reason, pnl_pct, strategy_source
FROM portfolio_positions
WHERE status = 'closed'
ORDER BY close_date DESC
LIMIT 20;
```

### 每日净值曲线
```sql
SELECT nav_date, total_value, cash, position_count,
       daily_return, cumulative_return, excess_return
FROM portfolio_nav
ORDER BY nav_date;
```

### 胜率统计
```sql
SELECT
    COUNT(*) AS total_trades,
    COUNT(CASE WHEN pnl_pct > 0 THEN 1 END) AS wins,
    COUNT(CASE WHEN pnl_pct <= 0 THEN 1 END) AS losses,
    ROUND(AVG(CASE WHEN pnl_pct > 0 THEN pnl_pct END) * 100, 2) AS avg_win_pct,
    ROUND(AVG(CASE WHEN pnl_pct <= 0 THEN pnl_pct END) * 100, 2) AS avg_loss_pct
FROM portfolio_positions
WHERE status = 'closed';
```

### 按策略统计
```sql
SELECT strategy_source,
       COUNT(*) AS trades,
       ROUND(AVG(pnl_pct) * 100, 2) AS avg_pnl_pct,
       COUNT(CASE WHEN pnl_pct > 0 THEN 1 END) AS wins
FROM portfolio_positions
WHERE status = 'closed'
GROUP BY strategy_source;
```

## 六、常见问题

### Q: 19:30 运行时数据还没更新完怎么办？
数据更新通常 3 分钟内完成（19:03 左右）。30 分钟的间隔足够。如果确实出现数据延迟，系统会使用前一个交易日的数据运行，不会报错，但当天的因子和信号会基于旧数据。

### Q: 非交易日会不会误操作？
不会。`run_daily.py` 检查数据库中最新交易日期，如果距今超过 3 天就自动跳过。周末和节假日都不会执行任何交易。

### Q: 想重置模拟盘从头开始怎么办？
```sql
DELETE FROM trade_log;
DELETE FROM portfolio_positions;
DELETE FROM portfolio_nav;
DELETE FROM signal_history;
```

### Q: north_flow_chg 和 north_days 因子什么时候有数据？
`north_hold_pct` 通过每日定时任务逐步积累（约 2700 只/天）。`north_net_buy` 由因子引擎通过连续两天的持仓占比差值计算。模拟盘运行约 5 天后 `north_flow_chg` 开始生效，20 天后 `north_days` 有完整数据。

### Q: 怎么查看某只股票为什么被过滤？
```sql
SELECT stock_code, filter_reason
FROM signal_history
WHERE trade_date = '2026-03-27' AND filter_reason IS NOT NULL;
```
